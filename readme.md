# УЛАД (Универсальный Логический Агрегат Данных)

**Полностью импортозамещённый формат конфигурационных файлов**

> Никаких англицизмов. Только русский язык.

---

## 🇷🇺 Философия

JSON — это скучно. YAML — слишком много отступов. XML — для корпоративных рабов. TOML — слишком по-западному.

**УЛАД** — создан для настоящих патриотов, которые хотят писать конфиги на великом и могучем.

---

## 📦 Установка

```bash
git clone https://github.com/stastpru/ulad
cp ulad_parser.py /ваш/проект/
```

Никаких зависимостей. Только Python и любовь к родине.

## ⚠️ Ограничения

- Кодировка файлов — **только UTF-8**
- Имена секций и ключей — **только кириллица**
- Первая строка — **обязательно** `УЛАД-СХЕМА:3.0`
- Латинские буквы в данных — **ЗАПРЕЩЕНЫ**
